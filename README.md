<h1 align="center">GiTUI</h1>


<h2 align="center">Overview</h2>

GitUI is an interactive Terminal User Interface designed to explore GitHub repositories graphically and efficiently. It allows developers to browse file structures and preview repository details directly from the command line, eliminating the need to use a web browser before cloning a project.



<h2 align="center">Core Features</h2>

* **Instant Search:** Locate repositories by name, user, or organization.
* **File Explorer:** Visualize folder structures and file contents in real-time using the GitHub API.
* **Streamlined Cloning:** Select any repository and clone it to your local directory with a single keystroke.
* **Developer Insights:** Quick access to repository statistics including stars, primary language, and descriptions.
* **Keyboard-Centric Navigation:** Fully optimized for terminal-based workflows to maximize productivity.
